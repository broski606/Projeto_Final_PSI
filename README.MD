<p>Para rodar este programa é necessário ter um servidor de base de dados ativo!!!</p>
<p>Pode usar o docker entre muitos.</p>
<p>Para instalar o container basta correr: docker run --name mysql -e MYSQL_ALLOW_EMPTY_PASSWORD=yes -p 3306:3306 -d mysql:latest</p>
<p>Depois basta correr os comandos SQL no ficheiro base_de_dados.sql</p>
